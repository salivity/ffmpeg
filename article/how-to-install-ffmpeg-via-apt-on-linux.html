How to Install FFmpeg via APT on Linux

This guide provides a straightforward, step-by-step walkthrough for installing FFmpeg on Linux systems using the Advanced Package Tool (APT) package manager. You will learn how to update your package lists, install the FFmpeg software, and verify that the installation was successful. This method applies to popular Debian-based distributions, including Ubuntu, Linux Mint, and Debian itself.

Step 1: Update Your Package Lists

Before installing any new software, it is best practice to update your local package index. This ensures your system knows about the latest available versions and security patches.

Open your terminal and run the following command:

sudo apt update

You will be prompted to enter your administrator password to authorize the update.

Step 2: Install FFmpeg

Once your package lists are refreshed, you can install FFmpeg and its required dependencies with a single command.

Execute the following in your terminal:

sudo apt install ffmpeg

The terminal will display the packages to be installed and ask for your confirmation. Press Y and then Enter to proceed with the installation. APT will handle downloading and configuring all the necessary files.

Step 3: Verify the Installation

After the installation process completes, you should verify that FFmpeg is correctly installed and accessible from your command line.

Check the installed version by running:

ffmpeg -version

If the installation was successful, the terminal will print the version number, configuration details, and copyright information for FFmpeg. You are now ready to use FFmpeg for audio and video conversion, recording, and streaming.
