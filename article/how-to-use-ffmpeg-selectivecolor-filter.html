How to Use FFmpeg selectivecolor Filter

This article explains how to use the FFmpeg selectivecolor filter to target and adjust specific color ranges, such as cyans, magentas, yellows, and blacks. You will learn the basic filter syntax, understand how CMYK adjustments work on digital video, and explore practical examples to shift and fine-tune colors in your video files.

Understanding the selectivecolor Filter

The selectivecolor filter in FFmpeg mimics the selective color adjustment tool found in professional image editing software like Adobe Photoshop. It allows you to select a specific color range and adjust the amount of Cyan, Magenta, Yellow, and Black (CMYK) within that range.

The filter is highly useful for color grading, color correction, and creative color shifts without affecting the entire image.

Color Ranges You Can Target

You can target nine distinct color ranges: * reds * yellows * greens * cyans * blues * magentas * whites * neutrals * blacks

CMYK Adjustment Values

For each targeted color range, you specify adjustments for Cyan (c), Magenta (m), Yellow (y), and Black (k). These values range from -1.0 to 1.0 (representing -100% to +100% adjustment).

Basic Syntax

The basic syntax for the filter is:

-vf "selectivecolor=range='c m y k'"

You can also specify the correction_method as either absolute (default) or relative: * Absolute: Adds the exact value directly to the color channel. * Relative: Adjusts the color relative to the existing amount of that color in the pixel, resulting in a more natural blend.

-vf "selectivecolor=correction_method=relative:cyans='0.2 -0.1 0 0'"

Practical Examples

Example 1: Adjusting Cyans and Magentas

To boost the Cyan channel in cyan-colored objects and reduce Yellow in magenta-colored objects, use the following command:

ffmpeg -i input.mp4 -vf "selectivecolor=cyans='0.3 0 0 0':magentas='0 0 -0.4 0'" -c:a copy output.mp4
  • cyans='0.3 0 0 0': Increases cyan by 30% inside cyan regions.
  • magentas='0 0 -0.4 0': Decreases yellow by 40% inside magenta regions, making them appear cooler and more pink.

Example 2: Creating Teal and Orange Look

You can shift blues toward cyan and reds toward yellow to create a cinematic teal-and-orange effect:

ffmpeg -i input.mp4 -vf "selectivecolor=blues='0.4 -0.2 -0.2 0':reds='-0.2 0.1 0.4 0'" -c:a copy output.mp4
  • blues='0.4 -0.2 -0.2 0': Increases cyan, while decreasing magenta and yellow in blue areas, pushing them toward teal.
  • reds='-0.2 0.1 0.4 0': Decreases cyan, while increasing magenta and yellow in red areas, pushing them toward orange.

Example 3: Modifying Neutrals for Overall Color Temperature

To warm up or cool down the midtones (neutrals) of your video without clipping the highlights or crushing the shadows:

ffmpeg -i input.mp4 -vf "selectivecolor=neutrals='-0.1 0 0.15 0'" -c:a copy output.mp4
  • neutrals='-0.1 0 0.15 0': Decreases cyan (-10%) and increases yellow (+15%) in neutral tones, applying a warm, golden cast to the midtones.
